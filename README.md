# Everhaul Website

## How to deploy on Vercel

1. Download and unzip this folder.
2. Go to https://vercel.com/new
3. Drag and drop the entire `everhaul-website` folder.
4. Click Deploy.
5. After it goes live, connect your domain inside Vercel > Project > Settings > Domains.

## Local preview

```bash
npm install
npm run dev
```

Then open http://localhost:3000
